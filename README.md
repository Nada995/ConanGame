# جريمة في ليلة الزفاف

لعبة تفاعلية بسيطة لفعالية العيد في الدوام.  
المشروع جاهز للرفع على GitHub Pages.

## الملفات

- `index.html` هي الصفحة الرئيسية.
- `style.css` تنسيق وتصميم الصفحة.
- `script.js` بيانات اللعبة والتفاعل.

## طريقة التشغيل محليًا

افتحي ملف `index.html` مباشرة في المتصفح.

## طريقة الرفع على GitHub Pages

1. أنشئي Repository جديد في GitHub.
2. ارفعي الملفات الثلاثة: `index.html`, `style.css`, `script.js` مع `README.md`.
3. ادخلي إلى Settings.
4. اختاري Pages.
5. في Source اختاري Deploy from a branch.
6. اختاري Branch: `main` و Folder: `/root`.
7. اضغطي Save.

بعد دقائق سيظهر رابط اللعبة.

## تعديل القصة

افتحي ملف `script.js` وعدلي البيانات داخل:

```js
const GAME_CONFIG = { ... }
```

يمكنك تغيير القاتل من السطر:

```js
killer: "jood"
```

مع مراعاة استخدام نفس `id` الموجود في قائمة المشتبه بهم.
